# Data Rights and Attribution

This dataset contains **metadata records** collected from **Trove (National Library of Australia)**.

All rights to the original materials and their associated metadata belong to **Trove / National Library of Australia** and to the contributing institutions represented in Trove.

The dataset distributed in this repository contains only descriptive metadata and links to the original Trove records.

The metadata was programmatically collected for **Open Data Armenia** in order to identify, collect, document, and preserve Armenian cultural heritage represented in international archives and digital collections.

Users of this dataset should acknowledge:

**Trove (National Library of Australia)**

Links to the original Trove records are included in the dataset so that users can consult the authoritative source directly.
